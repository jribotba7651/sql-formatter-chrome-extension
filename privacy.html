Privacy Policy

SQL Formatter & Beautifier - Format SQL Queries

Last Updated: November 2024

Summary: This extension does not collect, store, or transmit any personal user data. All processing happens locally in your browser.

Data Collection

The SQL Formatter & Beautifier extension does not collect, store, or transmit any personal user data. We are committed to protecting your privacy and ensuring that your data remains yours alone.

Local Processing

All data entered in the extension (such as SQL queries) is processed entirely locally within your browser. No information is sent to external servers, and no data leaves your device.

Storage

The extension uses Chrome's local storage API only to save your preferences, including:

  • Indentation settings
  • Capitalization style preferences
  • Other formatting options

These settings remain completely private and are stored locally on your device. They are never transmitted to any external servers or third parties.

Permissions

The extension requests the following Chrome permissions:

  • storage: To save your formatting preferences locally
  • clipboardWrite: To allow you to copy formatted SQL to your clipboard

These permissions are used solely for the functionality described and nothing else.

Third-Party Libraries

The extension may use external SQL formatting libraries for syntax highlighting and formatting. These libraries run locally in your browser and do not share, track, or transmit any data.

No Analytics or Tracking

This extension does not use any analytics, tracking, or monitoring tools. We do not track your usage, collect statistics, or monitor your activity in any way.

Changes to This Policy

If we make any changes to this privacy policy, we will update this page accordingly. We recommend checking this page periodically for any updates.

Contact

If you have any questions or concerns about this privacy policy, please visit our GitHub repository to open an issue or contact us.

Agreement

By using this extension, you agree that no personal or sensitive data will be collected, stored, or transmitted at any time.
