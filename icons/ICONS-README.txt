ICONS NEEDED
============

You need to add 4 icon files to this folder for the Chrome extension to work properly:

Required files:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

EASIEST OPTION - Use the Icon Generator:
=========================================
1. Open the file: create-icons.html (in the root folder)
2. Click "Download All Icons"
3. Move the 4 downloaded PNG files to THIS folder (icons/)
4. Done! ðŸŽ‰

Alternative Options:
===================
1. Use an icon generator website like:
   - https://www.favicon-generator.org/
   - https://redketchup.io/icon-converter

2. Find a free icon at:
   - https://www.flaticon.com/ (search "database" or "sql")
   - https://icons8.com/

3. Copy the icons from the Jibaro extension temporarily

4. Create your own using any image editor (GIMP, Photoshop, etc.)

Note: The extension will work without icons, but Chrome will show a default placeholder.
For the best experience, use the create-icons.html generator!
