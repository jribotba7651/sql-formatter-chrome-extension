Want to try it first? Open `test-formatter.html` in your browser to test the formatter without installing the extension.

## Troubleshooting

### Extension not appearing?
- Make sure Developer mode is enabled in `chrome://extensions/`
- Check that all files are in the correct folder
- Try refreshing the extensions page

### Icons not showing?
- Make sure you have all 4 PNG files in the `icons/` folder
- File names must be exactly: icon16.png, icon32.png, icon48.png, icon128.png
- The extension will still work, Chrome will just show a default icon

### Formatter not working?
- Open the test-formatter.html file to test if it's a code issue
- Check the browser console for errors (F12)
- Report the issue on GitHub

## Uninstalling

1. Go to `chrome://extensions/`
2. Find "SQL Formatter & Beautifier"
3. Click "Remove"
